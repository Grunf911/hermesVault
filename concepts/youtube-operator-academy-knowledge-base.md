# YouTube Creator Master Framework

## Purpose
A complete operating framework for YouTube content strategy, covering ideas, titles, thumbnails, and hooks.

## Core operating principle
The winning sequence is:
1. Idea
2. Title
3. Thumbnail
4. Hook

The base idea matters most. Packaging only works if the underlying idea is proven and strong.

## Video ideation system
### Main principles
- Ideas are more important than everything else
- Do not invent ideas from scratch; model proven formats and adapt them
- Validate before creating
- Every idea should include topic, format, and angle

### Outlier research
- Compare video views to channel average to find outliers
- 3x is minimum, 5x is ideal, 10x is gold mine
- Small-channel outliers under 100k subscribers are especially valuable
- Study a Dream 10 and a Model 10

### 7 Lego Bricks
Break winning videos into:
- Topic
- Angle
- Hook structure
- Story structure
- Visual format
- Key visuals
- Audio

### Idea filters
- Use the 50 in 5 method: 10 ideas per day for 5 days
- Validate with six checks: feasibility, new audience interest, existing audience interest, view potential, brand alignment, packageability
- Score validated ideas for excitement, effort, and confidence

### Winning formats
- Tutorials
- Challenges
- Breakdowns
- Commentaries
- Vlogs
- Interviews

## Title system
### Three click-worthy emotions
- Curiosity
- Desire
- Fear

### ACE framework
- Attention
- Clarity
- Engagement

### Title formulas
- Listicle
- Warning
- Transformation
- How-to benefit
- Authority secret
- Comparison
- Open loop
- Audience callout
- Time challenge
- Counterintuitive

### Title rules
- Keep titles under 55 characters when possible
- Front-load the most important words
- Avoid the word “and”
- Avoid generic filler like tips, tricks, and hacks
- Do not repeat thumbnail text
- Prefer active, direct language
- Make titles benefit-driven

### Power words
Use emotional, authority, urgency, exclusivity, curiosity, and desire words strategically.

## Thumbnail system
### Thumbnail categories
- Collage
- Showcase
- Reaction
- Split Frame
- Single Action

### 3 C’s framework
- Contents: maximum 3 elements
- Composition: clear hierarchy and rule of thirds
- Contrast: internal, competitive, and platform contrast

### Color science
High-performing combinations include:
- Orange + blue
- Yellow + purple
- Red + black
- Green + pink
- Cyan + orange

### Thumbnail rules
- Use faces in most cases
- Keep the bottom-right corner clear for the timestamp
- Make the main subject dominate the frame
- Make text readable at mobile size
- Create a visual that adds new information, not duplicate title text
- Design the thumbnail before filming

### Common mistakes
- Too many elements
- Tiny text
- Unknown face when a clearer focal point would work better
- Copying competitors too closely
- Dull colors
- Title and thumbnail saying the same thing
- Clickbait mismatch

## Hook system
### Five-step retention formula
1. Interest spark in the first 4–8 seconds
2. Pattern interrupt in 1–5 seconds
3. Twist and value in 1–2 sentences
4. Credibility in 1 sentence
5. Compression: keep it short, punchy, and clear

### Three-way alignment
- Spoken hook
- Visual hook
- Text hook

If those do not align, viewers get confused and leave.

### Hook archetypes
- Curiosity
- Storytelling
- Challenge
- Negative
- Comparison
- Visual

### Hook killers
- Delay
- Confusion
- Irrelevance
- Disinterest

### Hook flow
Use hook → deliver → hook → deliver throughout the video.

## Checklists
The document’s checklists can be used as pre-publish gates for:
- Idea research and validation
- Title design
- Thumbnail design
- Hook generation

## Practical workflow
1. Find a proven outlier idea
2. Turn it into a clear topic-format-angle package
3. Write a title with an emotion and formula
4. Design a thumbnail with contrast and one clear focal point
5. Write a hook that matches title and thumbnail
6. Run the checklists before filming or publishing
